# docuquery-ai
RAG-based AI document assistant using FastAPI, HuggingFace embeddings and Ollama.

#description:
A smart document query system that lets users upload documents, index their content, and ask questions in natural language, returning AI-generated answers based on the uploaded documents.

#Key Features:
Upload PDFs and extract text automatically
Convert text into embeddings for semantic search
Query documents using natural language and get AI-generated responses
FastAPI REST API with Swagger UI for testing
Handles multiple documents and queries efficiently

#Tech Stack / Tools Used:
Python 3.12 – main programming language
FastAPI – REST API and Swagger UI
Pydantic – request validation
PyPDF2 – PDF text extraction
Sentence-Transformers (Hugging Face) – embeddings for semantic search
CloudAI / Ollama – LLM for generating answers
VS Code – development environment
Git & GitHub – version control and project hosting
Virtual Environment (venv) – dependency management

#Impact / Learning:
Built a fully functional RAG (Retrieval-Augmented Generation) system
Gained hands-on experience with AI pipelines, embeddings, and LLMs
Learned production-level API design, error handling, and testing with Swagger
Prepared groundwork for CI/CD integration and scalable deployments
